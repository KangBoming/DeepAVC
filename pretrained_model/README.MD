Pre-trained Models and Feature Extraction Tools

The pre-trained model files and necessary feature extraction models for compounds and proteins are provided on our web server. You can access the resources via the following link: http://www.cuilab.cn/deepavc

The available models and tools include:

* DeepPAVC: Phenotype-based Antiviral Prediction

This model is designed to predict potential antiviral compounds based on phenotype data.

* DeepTAVC: Target-based Antiviral Prediction

Focuses on predicting antiviral compounds by utilizing target information, assessing interactions between compounds and specific targets.

* CADTI: General Compound-Protein Interaction Prediction

A general-purpose model for predicting potential interactions between compounds and proteins, supporting a wide range of molecular data types.

* KPGT: Compound Feature Extraction

A pre-trained model for extracting compound features, enabling the retrieval of informative descriptors from chemical structures.

* ESM-2: Protein Feature Extraction

Utilizes the pre-trained ESM-2 model to extract high-level features from protein sequences, supporting downstream analyses.
